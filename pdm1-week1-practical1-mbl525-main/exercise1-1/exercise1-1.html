Exercise 1.1

Your sketch:

Instructions

Create a sketch that is 600 pixels by 400 pixels. The sketch should have a black background.

Store the 100 most recent mouse coordinates, and use these coordinates to draw circles that appear to follow the path of the mouse. You may choose any size and fill colour you like but the fill should appear to fade out from the circle at the most recent coordinate to the circle at the oldest coordinates. Take a look at the video of the expected output below to get a sense of what you're being asked to do.

You only need to edit sketch.js.

The key to this exercise is making good use of arrays and a for loop. You will need one array to store the mouseX values and another array to store the mouseY values. Each time draw() runs, add the current mouseX and mouseY values to the arrays.

In order to limit the size of the array to 100 and keep track of the mouse coordinates in order, you will need to use some of JavaScript's handy array methods. Take a look at these:

  • unshift()
  • pop()
  • slice()

To make the circles fade, use the index in the for loop and perform a simple calculation to work out an alpha value for each circle.
